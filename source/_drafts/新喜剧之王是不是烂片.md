# 《新喜剧之王》是不是烂片

Created By: Loki Chen
Last Edited: Aug 20, 2019 4:17 PM

这个电影的主题是什么？总结了一下网上的看法，主要分三种：

##A：女主对于演戏这件事坚持不懈，努力奋斗，最终取得了事业上的成功。
持这种观点的观众，大多会觉得今时今日，还在提倡 “努力&奋斗，就能取得成功” 这种鸡汤，实在有够落后和滥俗。所以《新喜剧之王》可谓一个毫无新意，炒老梗的圈钱之作。

##B：只要坚持不放弃做一件事，这样的人生已经是另一种成功。
我最初在看完电影的时候，就是得出这样的总结的。当然这个总结也很显而易见，在电影里是由王宝强的角色台词，直接说出来的。

##C：第三种这里先放在文章后半部说。

如果是按A的理解，这部片的确有够俗套，但这仅仅是最表面的A故事。而又有什么证据支持B的理解呢？首先看过电影的人都会感觉女主最后的成功，过于轻松。包括演员海选晋级，推开那扇门的时候，射入来的光线亮得极其不真实。我自己的理解是，其实在女主骑摩托出车祸的那一刻，后面的都已经是非现实。残酷点说，也许女主已经死了(或者昏迷)，后面成功的那段都只是女主的一个想象。

那有人会质疑有什么证据？还是一厢情愿过度解读？可以分以下几个方面来说。

以往电影作品证实：如果有去留意《长江七号》《功夫》等星爷电影，这种设计已经早就有了。

##第三种主题理解

这部片主要是对演技这件事进行了探讨。也许这也是为什么大部分人，get不到同感的原因。这个暗线实在不像大多数商业片那么贴近普通人。

我们会看到很多悬疑片，最终可能是讲亲情；或者看很多喜剧片，其实内核讲的是爱情。不管是亲情还是爱情，大多都还是普通人生活里能感受的东西。这样除了故事本身讲圆讲顺，我们在回想的时候都会多几分感同身受，也对故事的说服力提供了更大的帮助。

而这部片，表面是一个“努力-奋斗-成功”的鸡汤，但内核是在说“演技”这个事。这个就不是大多数人日常生活里贴近的东西了。所以回顾整个电影，实际上是双主角。

女主从一开场就教路人演戏，天天看《演员的自我修养》，自以为能力和努力程度都很高，迟早一天能当女主角。而男主从童星出名发迹，老早就成为国际巨星，但实际上也是毫无演技。但由于要维持自身的巨星形象，总要装有料，受不了旁人对他的一点点指责和轻视。实际上，这电影一开始就是在呈现这两个不会演戏的人。

那他们是什么时候发生转变的呢？

王宝强，是因为被剧组装鬼，吓出了他最真实的反映，最没有防备。（所以这里并不是主要讽刺演艺圈的人，可以因为一段偶然的视频无端走红。）大家喜欢他那段视频，主要是因为他的反应真实，演得好。

而女主是什么时候才会演戏的呢？是在被合约男友欺骗后，人生经历了剧痛，才学会的。总的来说，他们两人的蜕变都是来自生活的体验，是最贴近自己内心的一次体验。

所以，女主在得知男友是欺骗她的时候，依然把帮他凑的几万块钱，交给她男友。这个设计的背后逻辑是什么呢？
